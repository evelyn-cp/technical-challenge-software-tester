# Technical Challenge - Test Automation Project

This repository contains an automated testing project developed as part of a technical challenge.
The solution covers API testing and web UI testing, following best practices for test automation using Robot Framework.

## Scope

The project includes automated tests for:

- API testing: HTTP methods: GET, POST, and DELETE
- Web UI testing: Browser-based interaction with a web application

## Setup
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
rfbrowser init
```

## Execution
```bash
robot --outputdir results tests/
```
